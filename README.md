# Stealth Reader

## English

**Stealth Reader** is a browser extension designed for a discreet and optimized reading experience on Archive of Our Own (AO3).

### Features
- Simple popup interface to read AO3.
- Clean and optimized layout natively accessible from your browser toolbar.

### How it Works
The extension works by fetching the story content directly from Archive of Our Own (AO3) in the background. It retrieves the HTML text and renders it cleanly within the extension's popup window. This allows you to read your stories discreetly without needing to keep the actual AO3 website visible or open in an active tab.

### Installation Guide (Google Chrome, Brave, Edge)

1. Download or clone this repository to your local machine.
2. Open your browser and navigate to the extensions management page:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. Toggle the **Developer mode** switch, usually located in the top right corner.
4. Click on the **Load unpacked** button.
5. Select the directory where you saved the extension files (the folder containing the `manifest.json` file).
6. The extension is now installed and ready to use. You can pin it to your browser toolbar for easier access.

## Français

**Stealth Reader** est une extension de navigateur conçue pour offrir une expérience de lecture discrète et optimisée sur Archive of Our Own (AO3).

### Fonctionnalités
- Interface de lecture simplifiée via popup.
- Mise en page épurée et optimisée, accessible directement depuis la barre d'outils de votre navigateur.

### Comment ça marche
L'extension fonctionne en récupérant ("fetching") le contenu de l'histoire directement depuis Archive of Our Own (AO3) en arrière-plan. Elle extrait le texte HTML et l'affiche proprement dans la fenêtre popup de l'extension. Cela vous permet de lire vos histoires avec une plus grande discrétion, sans avoir besoin de garder le site AO3 visible ou ouvert dans un onglet actif.

### Guide d'installation (Google Chrome, Brave, Edge)

1. Téléchargez ou clonez ce dépôt sur votre machine locale.
2. Ouvrez votre navigateur et accédez à l'URL de gestion des extensions :
   - Chrome : `chrome://extensions/`
   - Edge : `edge://extensions/`
   - Brave : `brave://extensions/`
3. Activez l'interrupteur **Mode développeur** (Developer mode), généralement situé dans le coin supérieur droit.
4. Cliquez sur le bouton **Charger l'extension non empaquetée** (Load unpacked).
5. Sélectionnez le dossier dans lequel vous avez enregistré les fichiers de l'extension (le dossier qui contient le fichier `manifest.json`).
6. L'extension est désormais installée. Vous pouvez l'épingler à la barre d'outils de votre navigateur pour y accéder plus facilement.
